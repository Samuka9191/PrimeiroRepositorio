# 6.2 Comandos Git

## 6.2.1 Init
Inicia um repositório Git no projeto.

## 6.2.2 Add
Adiciona arquivos para preparar o salvamento.

## 6.2.3 Status
Mostra os arquivos alterados.

## 6.2.4 Config
Configura usuário e e-mail do Git.

## 6.2.5 Commit
Salva as alterações no histórico.

## 6.2.6 Log
Mostra o histórico de commits.
